# Arrhythmia Classifier 🫀🤖

Machine Learning model to classify cardiac arrhythmias using ECG signals from the MIT-BIH Database.

## Results
- 99% accuracy on test data
- Detects Normal, Atrial, and Ventricular beats
- Random Forest with 100 estimators

## Technologies
- Python 3.13
- Scikit-learn, Pandas, NumPy
- Matplotlib, Seaborn, WFDB

## Data Source
MIT-BIH Arrhythmia Database - PhysioNet

## Conclusion

This project demonstrates that Machine Learning can effectively classify cardiac arrhythmias from ECG signals with 99% accuracy. Using a Random Forest model trained on real patient data from the MIT-BIH Database, the classifier successfully distinguishes between normal beats, atrial premature contractions, and ventricular contractions.

This type of model has real-world applications in wearable medical devices and remote patient monitoring systems, where early arrhythmia detection can be critical for patient outcomes.
